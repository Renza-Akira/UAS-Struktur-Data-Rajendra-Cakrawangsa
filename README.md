# UAS-Struktur-Data-Rajendra-Cakrawangsa
Mini WhatsApp adalah aplikasi pesan sederhana berbasis Python dengan penyimpanan data menggunakan file CSV. Aplikasi ini menyediakan fitur register, login, kirim pesan, inbox, melihat seluruh chat, menghapus pesan, dan logout.

Disini juga saya membangun Asisten Rara Ai dengan natural tanpa menggunakan api key supaya mudah digunakan dan pastinya gratis. Untuk itu saya sudah merangkum tata cara penggunaan Apk WhatsApp Mini dan juga Asisten Rara Ai menggunakan Struktur Database flat file dari python CSV Sebagai pengumpulan UAS Struktur Data.
